# RussianAiCup2019
Открытое соревнование по программирвоанию искуственного интеллекта.

#### Мой результат:

* Песочница - 2502 / 394
* Раунд 1 - 74 / 216
* Раунд 2 - 103 / 258

## How It Works

1. Запустить приложение /AppForLocalTest/aicup2019.exe
2. Запустить компиляцию проекта в среде. (В приложении "Ожидание подключения" изменится на "Игрок подключился" )
3. В приложении нажать "Старт"

p.s. приложение использует порт 31001.
